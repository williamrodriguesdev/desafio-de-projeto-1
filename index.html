H1 a H6

(inicio)

As tags h1 até h6 são usadas para definir cabeçalhos HTML. H1 define o cabeçalho mais importante.
H6 define o título menos importante.
Nota: Use apenas um h1 por página - isso deve representar o cabeçalho/assunto principal de toda a página. Além disso, não pule os níveis de títulos - comece com h1, depois use h2, e assim por diante.

P

(inicio)

A p tag define um parágrafo. Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada p elemento.

mark

(inicio)

A mark tag define o texto que deve ser marcado ou destacado.

small

(inicio)

A small tag define um texto menor (como direitos autorais e outros comentários laterais).
Dica: Esta tag não está obsoleta, mas é possível obter um efeito mais rico (ou o mesmo) com CSS.

i

(inicio)

A i tag define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico . A tag é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc. Use o i elemento somente quando não houver um elemento semântico mais apropriado, como:

  • em (texto enfatizado)
  • strong (texto importante)
  • mark (texto marcado/destacado)
  • cite (o título de um trabalho)
  • dfn (um termo de definição)

u

(inicio)

A tag u representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. O conteúdo interno geralmente é exibido com um sublinhado.
Dica: Evite usar o u elemento que pode ser confundido com um hiperlink!

strong

(inicio)

A strong tag é usada para definir o texto com forte importância. O conteúdo interno geralmente é exibido em negrito .

ol

(inicio)

A tag ol define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

ul

(inicio)

A ul tag define uma lista não ordenada. Use a ul tag junto com a tag li para criar listas não ordenadas.

li

(inicio)

A li tag define um item de lista. A li tag é usada dentro de listas ordenadas ( ol ), listas não ordenadas ( ul ) e em listas de menu ( menu ).Em ul e menu, os itens da lista geralmente serão exibidos com marcadores. Em ol, os itens da lista geralmente serão exibidos com números ou letras.

a

(inicio)

A tag define um hiperlink, que é usado para vincular de uma página a outra. O atributo mais importante do a elemento é o atributo href , que indica o destino do link. Por padrão, os links aparecerão da seguinte maneira em todos os navegadores:

  • Um link não visitado é sublinhado e azul
  • Um link visitado é sublinhado e roxo
  • Um link ativo é sublinhado e vermelho

hr

(inicio)

A hr tag define uma quebra temática em uma página HTML.

O hr elemento geralmente é exibido como uma régua horizontal usada para separar o conteúdo (ou definir uma alteração) em uma página HTML.

sub

(inicio)

A sub tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O. Dica: Use a tag sup para definir o texto sobrescrito.

sup

(inicio)

A sup tag define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW [1] .
Dica: Use a tag sub para definir o texto subscrito.

blockquote

(inicio)

A blockquote tag especifica uma seção citada de outra fonte. Os navegadores geralmente indentam blockquote elementos .
